| 값 | 의미 | 기본값 |
|---|:---:|---:| <- 이부분이 헤더와 바디를 나눠주는 선
 - 헤더 셀을 구분할 때 3개 이상의 - 기호가 필요
 - 헤더 셀을 구분하면서 :기호로 내용 정렬이 가능함 ( --- => 왼쪽정렬, :---: => 중앙정렬, ---: => 오른쪽 정렬  )
 - 양 옆의 |는 생략 가능


| 값 | 의미 | 기본값 |
|---|:---:|---:|
|`static`|유형(기준) 없음/ 배치 불가능|`static`|
|`relative`|요소 **자신**을 기준으로 배치|
|`absolute`|위치 상 **_부모_(조상)요소**를 기준으로 배치|
|`fixed`|**브라우저 창**을 기준으로 배치|
